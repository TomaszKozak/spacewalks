#Specwalks

##Overview
statistical summaries of NASA extravehicular

## Features
Key features of Spacewalks:
- Generates a CSV table of summary statistics of extravehicular activity and crew sizes
- Generates a line polot to show the cumulative duration of space walks over time

## Pre-requisite
Spacewalks was developed using Python 3.12

T install and run Spacewalks you will need to have Python >-3.12 installed.
You will also need the following libraries:
- [Numpy] (https://www.numpy.org/)
- [Matplotlib] (https://matplotlub.org/stable/index.html)
- [pytest] (https://docs.pytest.org/)
- [pandas] (https://pandas.pydata.org/)
